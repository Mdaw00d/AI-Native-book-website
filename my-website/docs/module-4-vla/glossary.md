# Glossary

## A-D

**Action Server**: ROS 2 component that executes long-running tasks with feedback

**COCO**: Common Objects in Context dataset with 80 object classes

**Depth Image**: Image where each pixel value represents distance from camera

## E-L

**LLM**: Large Language Model - neural network trained on text

**Nav2**: ROS 2 navigation stack for autonomous robot movement

## M-R

**MoveIt 2**: Motion planning framework for manipulation

**ROS 2**: Robot Operating System version 2

## S-Z

**STT**: Speech-to-Text conversion

**URDF**: Unified Robot Description Format

**VAD**: Voice Activity Detection

**Whisper**: OpenAI speech recognition model

**YOLO**: You Only Look Once - real-time object detection
