# Module 2 - The Digital Twin

## Overview

This module covers digital twin simulation for humanoid robots, combining Gazebo physics simulation, realistic sensor simulation, and Unity high-fidelity rendering.

## What You'll Learn

- **Chapter 1**: Physics-accurate simulation in Gazebo
- **Chapter 2**: Sensor simulation (LiDAR, depth cameras, IMU)
- **Chapter 3**: High-fidelity rendering in Unity
- **Chapter 4**: Integrated digital twin pipeline

## Prerequisites

- Basic understanding of robotics concepts
- Familiarity with ROS 2
- Linux development environment
