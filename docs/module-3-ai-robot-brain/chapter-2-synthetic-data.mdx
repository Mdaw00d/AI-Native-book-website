## Why Synthetic Data?

### The Real-World Data Problem

Training perception algorithms traditionally requires:
- **Expensive data collection**: Thousands of hours of real robot operation
- **Manual labeling**: Pixel-perfect segmentation masks, 3D bounding boxes
- **Limited diversity**: Hard to capture rare scenarios (poor lighting, occlusions)
- **Safety risks**: Cannot test dangerous edge cases (collisions, falls)

### The Synthetic Data Solution

Isaac Sim enables:
- **Automatic ground truth**: Perfect labels for segmentation, depth, poses
- **Infinite diversity**: Randomize lighting, textures, object positions
- **Safe edge cases**: Test failure scenarios without physical risk
- **Speed**: Generate 10,000 frames in hours vs. weeks of real data collection

<Callout type="success" title="Domain Randomization">
**Domain randomization** varies appearance during training so models generalize to real-world variations:
- Lighting: 2700K (warm) to 6500K (cool), 0-10,000 lux intensity
- Textures: Swap materials, randomize colors
- Poses: Randomize object positions ±50cm, rotations ±180°
- Camera params: Vary exposure, gain, white balance
</Callout>

## Step 3: Configure Virtual Sensor Suite

### Add Stereo Camera Pair

Stereo cameras enable depth perception for VSLAM (Chapter 3).

<CodeBlock language="python" title="add_stereo_cameras.py">
{`from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": False})

from omni.isaac.core import World
from omni.isaac.core.utils.stage import add_reference_to_stage
from omni.isaac.sensor import Camera
import omni.isaac.core.utils.prims as prim_utils
import numpy as np

world = World()

# Add warehouse and humanoid (from previous steps)
add_reference_to_stage("/Isaac/Environments/Simple_Warehouse/warehouse.usd", "/World/Warehouse")

# Stereo camera configuration
STEREO_BASELINE = 0.12  # 12cm between cameras (typical for humanoid head)
RESOLUTION = (640, 480)  # VGA resolution
FPS = 60  # Target frame rate
FOV_HORIZONTAL = 90.0  # Degrees

# Left camera
camera_left = Camera(
    prim_path="/World/CameraLeft",
    name="camera_left",
    position=[0.0, -STEREO_BASELINE/2, 1.5],  # 1.5m height (humanoid head)
    frequency=FPS,
    resolution=RESOLUTION,
    orientation=[1, 0, 0, 0]
)

# Right camera
camera_right = Camera(
    prim_path="/World/CameraRight",
    name="camera_right",
    position=[0.0, STEREO_BASELINE/2, 1.5],
    frequency=FPS,
    resolution=RESOLUTION,
    orientation=[1, 0, 0, 0]
)

# Set camera intrinsics (FOV)
for camera in [camera_left, camera_right]:
    camera.set_horizontal_aperture(20.955)  # Corresponds to 90° FOV
    camera.set_focal_length(24.0)  # mm

world.scene.add(camera_left)
world.scene.add(camera_right)
world.reset()

# Capture stereo pair
for _ in range(100):  # Warm-up frames
    world.step(render=True)

# Get synchronized stereo images
left_rgb = camera_left.get_rgba()[:, :, :3]  # Drop alpha channel
right_rgb = camera_right.get_rgba()[:, :, :3]

print(f"Left image shape: {left_rgb.shape}")  # (480, 640, 3)
print(f"Right image shape: {right_rgb.shape}")  # (480, 640, 3)

# Verify synchronization
left_timestamp = camera_left.get_current_frame()["rendering_frame"]
right_timestamp = camera_right.get_current_frame()["rendering_frame"]
drift = abs(left_timestamp - right_timestamp)
print(f"Timestamp drift: {drift * 1000:.3f} ms")  # Should be &lt;1ms

simulation_app.close()`}

## Step 4: Implement Domain Randomization

Domain randomization makes models robust to real-world variations.

### Randomize Lighting

<CodeBlock language="python" title="randomize_lighting.py">
{`from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": False})

from omni.isaac.core import World
from omni.isaac.core.utils.stage import add_reference_to_stage
import omni.isaac.core.utils.prims as prim_utils
import numpy as np
from pxr import Gf

world = World()
add_reference_to_stage("/Isaac/Environments/Simple_Warehouse/warehouse.usd", "/World/Warehouse")

# Create dome light
dome_light_path = "/World/DomeLight"
dome_light = prim_utils.create_prim(
    prim_path=dome_light_path,
    prim_type="DomeLight"
)

world.reset()

# Randomization loop
for i in range(10):  # Generate 10 variations
    # Randomize intensity (0-10,000 lux)
    intensity = np.random.uniform(500, 10000)
    prim_utils.set_prim_property(dome_light_path, "inputs:intensity", intensity)

    # Randomize color temperature (2700K warm to 6500K cool)
    # Convert Kelvin to RGB approximation
    kelvin = np.random.uniform(2700, 6500)
    if kelvin <= 6600:
        r = 1.0
        g = np.clip((kelvin / 100 - 60) * 0.39 + 0.1, 0, 1)
        b = np.clip((kelvin / 100 - 10) * 0.5, 0, 1) if kelvin > 2000 else 0
    else:
        r = np.clip(1.0 - (kelvin / 100 - 60) * 0.13, 0, 1)
        g = 1.0
        b = 1.0

    prim_utils.set_prim_property(dome_light_path, "inputs:color", Gf.Vec3f(r, g, b))

    # Step simulation and capture frame
    for _ in range(10):  # Warm-up
        world.step(render=True)

    print(f"Frame {i}: Intensity={intensity:.0f} lux, Kelvin={kelvin:.0f}K")

simulation_app.close()`}
</CodeBlock>

## Step 5: Generate Dataset with Ground Truth

### Complete Dataset Generation Script

<CodeBlock language="python" title="generate_dataset.py">
{`#!/usr/bin/env python3
from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": True})  # Headless for faster generation

from omni.isaac.core import World
from omni.isaac.core.utils.stage import add_reference_to_stage
from omni.isaac.sensor import Camera
from omni.isaac.core.objects import DynamicCuboid
import omni.isaac.core.utils.prims as prim_utils
import numpy as np
import cv2
import os
from datetime import datetime

# Configuration
NUM_FRAMES = 100  # Generate 100 frames (increase to 10,000 for production)
OUTPUT_DIR = f"dataset_{datetime.now().strftime('%Y%m%d_%H%M%S')}"
os.makedirs(f"{OUTPUT_DIR}/rgb", exist_ok=True)
os.makedirs(f"{OUTPUT_DIR}/depth", exist_ok=True)
os.makedirs(f"{OUTPUT_DIR}/segmentation", exist_ok=True)

# Create world
world = World()
add_reference_to_stage("/Isaac/Environments/Simple_Warehouse/warehouse.usd", "/World/Warehouse")

# Add camera
camera = Camera(
    prim_path="/World/Camera",
    position=[0, 0, 1.5],
    frequency=60,
    resolution=(640, 480)
)
world.scene.add(camera)

# Add random objects
for i in range(20):
    box = DynamicCuboid(
        prim_path=f"/World/Box_{i}",
        position=[np.random.uniform(-5, 5), np.random.uniform(-5, 5), 1.0],
        size=np.random.uniform(0.3, 0.8),
        color=np.random.rand(3)
    )
    world.scene.add(box)

# Create dome light for randomization
dome_light_path = "/World/DomeLight"
prim_utils.create_prim(dome_light_path, "DomeLight")

world.reset()

print(f"Generating {NUM_FRAMES} frames to {OUTPUT_DIR}/")

for frame_idx in range(NUM_FRAMES):
    # Randomize lighting
    intensity = np.random.uniform(500, 10000)
    prim_utils.set_prim_property(dome_light_path, "inputs:intensity", intensity)

    # Randomize object poses every 10 frames
    if frame_idx % 10 == 0:
        for i in range(20):
            box_path = f"/World/Box_{i}"
            new_pos = [
                np.random.uniform(-5, 5),
                np.random.uniform(-5, 5),
                np.random.uniform(0.5, 2.0)
            ]
            prim_utils.set_prim_property(box_path, "xformOp:translate", new_pos)

    # Step simulation
    for _ in range(10):  # Warm-up frames
        world.step(render=False)  # Headless: render=False

    world.step(render=True)  # Render final frame

    # Capture RGB image
    rgb = camera.get_rgba()[:, :, :3]  # Drop alpha
    rgb_bgr = cv2.cvtColor((rgb * 255).astype(np.uint8), cv2.COLOR_RGB2BGR)
    cv2.imwrite(f"{OUTPUT_DIR}/rgb/{frame_idx:06d}.png", rgb_bgr)

    # Capture depth image
    depth = camera.get_depth()
    depth_normalized = (np.clip(depth, 0, 10) / 10 * 65535).astype(np.uint16)  # 16-bit depth
    cv2.imwrite(f"{OUTPUT_DIR}/depth/{frame_idx:06d}.png", depth_normalized)

    # Capture semantic segmentation
    segmentation = camera.get_semantic_segmentation()
    cv2.imwrite(f"{OUTPUT_DIR}/segmentation/{frame_idx:06d}.png", segmentation.astype(np.uint8))

    # Print progress
    if (frame_idx + 1) % 10 == 0:
        print(f"Progress: {frame_idx + 1}/{NUM_FRAMES} frames ({(frame_idx+1)/NUM_FRAMES*100:.1f}%)")

print(f"\\nDataset generation complete!")
print(f"- RGB images: {OUTPUT_DIR}/rgb/")
print(f"- Depth images: {OUTPUT_DIR}/depth/")
print(f"- Segmentation: {OUTPUT_DIR}/segmentation/")

simulation_app.close()`}
</CodeBlock>

ROS 2 uses `rosbag2` for recording sensor data. We'll export Isaac Sim data to rosbag2 for use with Isaac ROS (Chapter 3).

### Export Script with rosbag2

<CodeBlock language="python" title="export_rosbag2.py">
{`#!/usr/bin/env python3
from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": True})

from omni.isaac.core import World
from omni.isaac.sensor import Camera
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, CameraInfo
from cv_bridge import CvBridge
import numpy as np
from datetime import datetime
import rosbag2_py

# Initialize ROS 2
rclpy.init()

# Create rosbag writer
bag_name = f"isaac_sim_{datetime.now().strftime('%Y%m%d_%H%M%S')}"
writer = rosbag2_py.SequentialWriter()
storage_options = rosbag2_py.StorageOptions(uri=bag_name, storage_id='sqlite3')
converter_options = rosbag2_py.ConverterOptions('', '')
writer.open(storage_options, converter_options)

# Create topics
topics = [
    rosbag2_py.TopicMetadata(
        name='/camera/left/image_raw',
        type='sensor_msgs/msg/Image',
        serialization_format='cdr'
    ),
    rosbag2_py.TopicMetadata(
        name='/camera/right/image_raw',
        type='sensor_msgs/msg/Image',
        serialization_format='cdr'
    )
]
for topic in topics:
    writer.create_topic(topic)

# Create world and cameras (stereo pair)
world = World()
camera_left = Camera(prim_path="/World/CameraLeft", position=[0, -0.06, 1.5], frequency=60, resolution=(640, 480))
camera_right = Camera(prim_path="/World/CameraRight", position=[0, 0.06, 1.5], frequency=60, resolution=(640, 480))
world.scene.add(camera_left)
world.scene.add(camera_right)
world.reset()

bridge = CvBridge()

print(f"Recording to {bag_name}.db3...")

for frame_idx in range(1000):  # Record 1000 frames
    world.step(render=True)

    # Get timestamp (nanoseconds since epoch)
    timestamp = int(world.current_time * 1e9)

    # Capture stereo pair
    left_rgb = camera_left.get_rgba()[:, :, :3]
    right_rgb = camera_right.get_rgba()[:, :, :3]

    # Convert to ROS Image messages
    left_msg = bridge.cv2_to_imgmsg((left_rgb * 255).astype(np.uint8), encoding='rgb8')
    right_msg = bridge.cv2_to_imgmsg((right_rgb * 255).astype(np.uint8), encoding='rgb8')

    # Set timestamps
    left_msg.header.stamp.sec = timestamp // 1_000_000_000
    left_msg.header.stamp.nanosec = timestamp % 1_000_000_000
    right_msg.header.stamp = left_msg.header.stamp  # Synchronized

    # Write to bag
    writer.write('/camera/left/image_raw', left_msg.serialize(), timestamp)
    writer.write('/camera/right/image_raw', right_msg.serialize(), timestamp)

    if (frame_idx + 1) % 100 == 0:
        print(f"Recorded {frame_idx + 1}/1000 frames")

print(f"\\nRecording complete: {bag_name}.db3")
print(f"Verify with: ros2 bag info {bag_name}")

writer.close()
simulation_app.close()
rclpy.shutdown()`}
</CodeBlock>

**Verify rosbag2 recording**:

### Verify Timestamp Synchronization

<CodeBlock language="python" title="check_sync.py">
{`import rosbag2_py
from rclpy.serialization import deserialize_message
from sensor_msgs.msg import Image

# Open bag
storage_options = rosbag2_py.StorageOptions(uri='isaac_sim_20251207_143025', storage_id='sqlite3')
converter_options = rosbag2_py.ConverterOptions('', '')
reader = rosbag2_py.SequentialReader()
reader.open(storage_options, converter_options)

left_timestamps = []
right_timestamps = []

while reader.has_next():
    (topic, data, timestamp) = reader.read_next()
    msg = deserialize_message(data, Image)

    msg_timestamp = msg.header.stamp.sec * 1e9 + msg.header.stamp.nanosec

    if topic == '/camera/left/image_raw':
        left_timestamps.append(msg_timestamp)
    elif topic == '/camera/right/image_raw':
        right_timestamps.append(msg_timestamp)

# Calculate drift
drifts = [abs(l - r) for l, r in zip(left_timestamps, right_timestamps)]
max_drift_ns = max(drifts)
avg_drift_ns = sum(drifts) / len(drifts)

print(f"Timestamp drift statistics:")
print(f"  Max drift: {max_drift_ns / 1e6:.3f} ms")
print(f"  Avg drift: {avg_drift_ns / 1e6:.3f} ms")
print(f"  Target: &lt;1 ms")

if max_drift_ns < 1e6:  # 1ms = 1,000,000 ns
    print("✅ PASS: Synchronization within spec (&lt;1ms)")
else:
    print("❌ FAIL: Drift exceeds 1ms threshold")`}
</CodeBlock>
