Project Specification: Physical AI Book Website

Intent:
Develop a premium, structured, Docusaurus-powered book-style website for the capstone topic Physical AI. The project begins with Module 1: The Robotic Nervous System (ROS 2), covering four core chapters:

ROS 2 Middleware

Nodes, Topics, and Services

Python-to-ROS Bridging via rclpy

URDF for Humanoids

All chapters will be clearly explained and authored in MDX for easy editing and scalability.

Pages & Layout:

Home: Introduction to Physical AI, overview of the book, and a call-to-action.

Module 1: Scrollable, MDX-based book layout with persistent sidebar, chapter navigation, and smooth reading experience.

About: Explains the purpose of Physical AI and the approach to teaching humanoid robotics.

Contact: Simple contact form saving data to localStorage.

Success Criteria:

Module 1 contains exactly four well-structured chapters in MDX.

Clean, book-like reading experience with persistent sidebar, smooth scroll, search, dark mode, and mobile-optimized layout.

Fully editable content via MDX.

Autogenerated navigation and polished, premium layout.

Fully functional Docusaurus build deployable within minutes.

Non-Goals:

No backend, authentication, user accounts, payments, comment systems, simulations, or execution of robotics code.
