## Summary

Develop a premium, structured, Docusaurus-powered book-style website for the capstone topic Physical AI. The project will leverage Docusaurus (MDX v3, dark/light mode, autogenerated sidebars), a modular chapter structure, and localStorage for client-side features. Emphasis is on a clean, minimal, book-like design with strong SEO and performance.

## Technical Context

**Language/Version**: Docusaurus (latest), MDX v3, React
**Primary Dependencies**: Docusaurus, `@docusaurus/preset-classic`, `@mdx-js/react`, (and other Docusaurus ecosystem plugins as needed)
**Storage**: localStorage (for contact form and reading position)
**Testing**: NEEDS CLARIFICATION
**Target Platform**: Web (mobile-first design)
**Project Type**: Web application (documentation site)
**Performance Goals**: Very fast performance (optimized bundle, plugin-based optimizations)
**Constraints**: No backend, authentication, user accounts, payments, comment systems, simulations, or execution of robotics code.
**Scale/Scope**: Book-style documentation website with multiple modules/chapters, starting with Module 1 (4 chapters). Designed to be versioning-ready.
